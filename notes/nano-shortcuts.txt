Ctrl+f - Forward a line
Ctrl+b - Backward a line
Ctrl+p - Up a line
Ctrl+n - Down a line
Ctrl+e - End of a line
Ctrl+a - Start of a line
Ctrl+v - Down a page
Ctrl+y - Up a page
Ctrl+Space - Forward a word
Meta+Space - Backward a word
Ctrl+o - Save a file
Ctrl+c - Show cursor
Ctrl+6 / Meta+a - Mark text
Meta+6 - Copy marked text
Ctrl+k - Cut marked text
Ctrl+k - Also deletes a line if not marked
Ctrl+w - Search (Ctrl+c - Cancel)
Alt+w - Search for the same string
Meta+u - Undo
Meta+e - Redo
Ctrl+4 - Search and replace
Esc+$ - Toggle wrap
Meta+g / Ctrl+_ - Jump to <line,column> (also `nano +<line_number> <file_name>`)
